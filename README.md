# file_sender
c++ file transfer program using socket communication

### study order

1. client to server (string)
2. client to server (file)
3. client to server (file) - multithread
4. client to server (multi file) - multithread

### 소켓

* 소켓 생성
* 소켓 바인딩 (bind)
* 연결 대기 (listen)
* 연결 수락 (accept)
* 데이터 송수신 (send, recv)
* 연결 종료 (close)
